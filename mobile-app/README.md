# Ikom Online Home – React Native Mobile App

This is the mobile app starter for Ikom Online Home, a marketplace connecting buyers, sellers, and riders.

## Folders

- `src/screens/`: App screens for authentication, buyers, sellers, riders
- `src/navigation/`: Navigation/Screens Stack
- `src/components/`: Reusable components

## To Start

1. Install dependencies
   ```sh
   npm install
   ```
2. Run the project (with Expo or React Native CLI)
   ```sh
   npm start
   ```

---
This is an MVP starter. Build your API, add UI/UX, and connect logic as you grow!
